# Dotfiles
Dotfiles for a bunch of useful stuff. 

- OS : Arch Linux
- WM : sway
- Bar: waybar
- Lockscreen : swaylock
- Launcher: wofi
- Shell : fish
- Terminal : foot
- Editor : nvim
